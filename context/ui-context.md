# UI Context

## Theme

**Bold Minimalism**. The design language is severely simple yet confident, utilizing stripped-back designs, vast expanses of negative space, and non-uniform grid structures. The interface acts as a meticulously controlled frame that does not visually compete with the highly dynamic, colorful motion graphics hosted within.

## Colors

Limited to a highly controlled 3-5 color palette to maintain focus on the portfolio pieces.

| Role            | CSS Variable       | Value/Description |
| --------------- | ------------------ | -------- |
| Page background | \--bg-base\        | Future Dust (dark blue with purple/grey undertones) |
| Surface         | \--bg-surface\     | Neutral contrast for case study cards |
| Primary text    | \--text-primary\   | High contrast (e.g., Off-white) |
| Muted text      | \--text-muted\     | Mid-tone neutral |
| Primary accent  | \--accent-primary\ | Vibrant Accent / Metallic Hue |

## Typography

Juxtapose typographic styles to establish sophisticated whimsy, brand recall, and strength.

| Role      | Font              | Variable      | Setting/Notes |
| --------- | ----------------- | ------------- | ------------- |
| Header / Display | Highly ornamental retro serif | \--font-display\ | Large display headers |
| Accent / Bold | Bold, condensed sans-serif | \--font-bold\ | 'Never Settle' brutalist impact |
| UI text   | System Font | \--font-ui\ | Heavy interface text to prevent loading jank |

## Border Radius

Embrace sharp or meticulously consistent border geometries to complement the non-uniform grid structures without making it look chaotic.

## Component Library

Webflow Native Components tied seamlessly to the Webflow Content Management System (CMS). This prevents recreating HTML/CSS logic across multiple pages.

## Layout Patterns

- **Homepage Hero**: Unobstructed auto-playing showreel taking the immediate viewport with zero clicks required.
- **Case Study Grids**: Highly scannable, non-uniform thumbnails with click-to-load video engagement.
- **Narrative Flow**: Deep, vertical/horizontal scroll case studies detailing project evolution.
- **Routing**: 'Strategic Friction' driven application/contact portals that force engagement with design philosophy rather than instant generic forms.
